# Weapon Coronas

Rockstar decided to remove the coronas from dropped weapons when San Andreas came around, it was a nice feature in III/VC that gave the game atmosphere. This mod adds coronas to weapons similar to III/VC, although not perfect, I think it suits the game and looks pretty good.
